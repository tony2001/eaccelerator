eAccelerator

php bytecode cache

Download this project as a .zip file Download this project as a tar.gz file

eAccelerator is a fast and simple bytecode cache for PHP. It stores the bytecode in shared memory (which is.. ahem.. shared across the processes) and retrieves this precompiled bytecode when needed, thus avoiding parsing and compiling stages for each cached file.

eAccelerator is dead, long live eAccelerator!

The eAccelerator project was never actively developed and it's apparently become orphaned about around 2010. But the code still works and hence we've decided to revive the project to share the results of our work with other people and get some feedback in the process.
